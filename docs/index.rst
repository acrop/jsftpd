Welcome to jsftpd documentation!
================================

jsftpd is an FTP server written in Javascript. It's feature set is

* Multi User capable
* Implicit/Explicit TLS
* Active/Passive mode for for data transfer
* Hooks for file operation

Quick Start
-----------

    npm install jsftpd
